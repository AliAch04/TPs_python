# Sélecteur de Variables Discriminantes

Application web interactive pour la sélection automatique des meilleures variables discriminantes entre plusieurs groupes.

## Fonctionnalités

- Import de fichiers CSV (Drag & Drop)
- 3 méthodes d'analyse :
  - ANOVA (Analyse de Variance)
  - Random Forest Feature Importance
  - Méthode combinée
- Visualisations interactives avec Chart.js
- Intrface moderne et responsive
- Dataset Iris intégré comme exemple
